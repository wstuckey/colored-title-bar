# Changelog

All notable changes to **Colored Title Bar** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/) and this
project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] — 2026-02-27

### Added

- **Randomize Color** command — apply a random, visually appealing colour to the title bar.
- **Pick Color Family** command — choose from 14 named hue presets (Red → Rose).
- **Reset to Default** command — remove all title-bar colour overrides.
- Automatic foreground-colour selection (white / black) for WCAG-compliant contrast.
- Inactive-window colour derivation (muted saturation + reduced-opacity foreground).
- `coloredTitleBar.colorOnStartup` setting for auto-randomisation on each launch.
- Full support for Dark, Light, and High Contrast themes.

## [0.1.0] — 2026-02-27

### Added

- **Randomize Color** command — apply a random, visually appealing color to the title bar.
- **Pick Color Family** command — choose from 14 named hue presets (Red → Rose).
- **Reset to Default** command — remove all title-bar color overrides.
- Automatic foreground-color selection (white / black) for WCAG-compliant contrast.
- Inactive-window color derivation (muted saturation + reduced-opacity foreground).
- `coloredTitleBar.colorOnStartup` setting for auto-randomisation on each launch.
- Full support for Dark, Light, and High Contrast themes.
